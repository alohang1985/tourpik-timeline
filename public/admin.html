Tourpik

1. 크롬에서 tourpik.com 카카오 로그인
2. F12 → Application → Cookies → www.tourpik.com
3. 모든 쿠키를 name=value; name2=value2 형태로 복사
또는 PHPSESSID 값만 입력해도 됩니다.

Adsun GPS

1. 크롬에서 dinhvi.adsun.vn 로그인
2. F12 → Console 에서 document.cookie 입력
3. token=XXXXX 부분의 값만 복사
또는 전체 쿠키 문자열을 붙여넣으면 자동 추출됩니다.
